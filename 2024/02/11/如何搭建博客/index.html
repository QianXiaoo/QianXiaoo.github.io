把博客部署到github上面

  1. 创建一个new repository，名称必须为***(为GitHub的用户名).github.io

  2. 安装git的部署插件(必须)

  3. 把blog文件放到vscode里面打开

    找到_config.yml

    repo后面加上github里仓库的地址

    部署到远端

    git config –global user.name”xxx” # xxx填github的用户名
    git config –global user.email”xxx@xxx” # xxx填github的邮箱地址

    hexo d

  2. 在_config.yml中将theme后面更改为yilia
