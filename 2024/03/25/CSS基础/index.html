div默认块级标签(霸道)，如果浮动起来就不一样。

如果你让标签浮动起来，就会脱离文档流。就是该标签不在依附于页面。

clear属性用于指定一个元素的两侧不允许浮动元素。当你在页面中使用浮动元素时，这可能会导致父元素的高度塌陷，影响布局。为了解决这个问题，可以在浮动元素之后插入一个具有 clear: both; 样式的空元素，该元素将强制在其下面的元素中清除浮动。

1.11 CSS知识点

1.11.1 hover知识点(伪类)

在 CSS 中，:hover 是一种伪类选择器，用于指定当用户将鼠标悬停在元素上时应用的样式。这允许你在用户与页面交互时改变元素的外观或行为。

使用 :hover 可以为任何可以接受用户交互的 HTML 元素定义不同的样式。例如，你可以改变链接的颜色，按钮的背景色，或者在悬停时显示一个元素。

案例解释：

  1. CSS样式：
    • .c1 类定义了一些基本的文本样式，包括颜色为红色，字体大小为18像素。
    • .c1:hover 类定义了当鼠标悬停在 .c1 元素上时应用的样式，包括改变文本颜色为绿色，字体大小增加到50像素。
    • .c2 类定义了一个具有固定高度、宽度和红色实线边框的方形块。
    • .c2:hover 类定义了当鼠标悬停在 .c2 元素上时应用的样式，包括改变边框颜色为绿色。
    • .download 类定义了一个初始状态下不可见的下载区域。
    • .app:hover .download 类定义了当鼠标悬停在 .app 元素上时，显示下载区域的样式。
    • .app:hover .title 类定义了当鼠标悬停在 .app 元素上时，标题颜色变为红色的样式。
  2. HTML内容：
    • <div class="c1">联通</div> 是一个使用 .c1 类的 <div> 元素，当鼠标悬停在上面时，文本颜色和字体大小会发生变化。
    • <div class="c2">广西</div> 是一个使用 .c2 类的 <div> 元素，当鼠标悬停在上面时，边框颜色会发生变化。
    • <div class="app"> 是一个包含两个子元素的 <div> 元素，当鼠标悬停在上面时，会显示 .download 区域和改变 .title 文本的颜色。
  3. 交互效果：
    • 当鼠标悬停在 .c1 元素上时，文本颜色变为绿色，字体大小增加。
    • 当鼠标悬停在 .c2 元素上时，边框颜色变为绿色。
    • 当鼠标悬停在 .app 元素上时，下载区域显示出来，标题文本颜色变为红色。

1.11.3 position

position 是 CSS 中用于定位元素的属性。通过 position 属性，你可以控制元素在页面上的布局方式，并且可以与其他属性结合使用来创建各种布局效果。

  • fixed
  • relative
  • absolute

fixed：固定在窗口的某个位置

案例：返回顶部

  • .back 类定义了一个元素样式，该元素在页面上的位置是固定的，即使页面滚动，它也会保持在视口中的固定位置。
  • width: 60px; 和 height: 60px; 分别设置了元素的宽度和高度为 60 像素。
  • border: 1px solid red; 添加了一个1像素宽度的红色实线边框。
  • right: 10px; 和 bottom: 50px; 分别设置了元素距离右侧和底部的定位偏移量。这意味着元素的右下角会距离视口的右侧10像素，底部50像素处。

这个类被命名为 .back，可能意味着这个元素是一个返回顶部的按钮或者类似的导航元素，因为它通常会被放置在页面的底部右侧，并且在用户滚动页面时保持可见性。

案例：对话框

z-index

z-index 是 CSS 属性，用于控制堆叠顺序，即确定元素在层叠上下文中的显示顺序。具体来说，z-index 属性的值越大，元素就越靠近用户视图。通常情况下，z-index 属性只在具有定位属性（如 position: absolute;、position: relative; 或 position: fixed;）的元素上生效。

水平居中：

left: 0; 和 right: 0;：这两个属性将对话框的左边缘和右边缘分别定位于视口的左边缘和右边缘。换句话说，它将对话框水平拉伸至视口的宽度，这样它的左右边缘都会与视口的对应边缘对齐。

margin: 0 auto;：这行代码实现了水平居中的关键。设置了 margin: 0 auto; 会使得对话框的左右外边距自动分配，从而将对话框水平居中于视口。因为 left: 0; 和 right: 0; 已经将对话框的宽度拉伸至视口的宽度，所以这个自动分配的外边距会使得对话框水平居中。

综合起来，left: 0; 和 right: 0; 属性将对话框水平拉伸至视口的宽度，而 margin: 0 auto; 属性将对话框水平居中。这样就实现了对话框在水平方向上的居中定位。

1.11.4 border

border: 1px solid red; 是 CSS 中设置边框样式的属性，它将会对元素的边框进行定义，具体解释如下：

  • 1px：设置边框的宽度为1像素。
  • solid：设置边框的样式为实线。实线是边框样式的一种，即边框由一系列矩形线段组成。
  • red：设置边框的颜色为红色。

如果只想要左边框，可以写：border-left，依次类推。
