Linux基础命令

1. 基础命令

.表示当前目录

..表示上级目录

cd切换当前目录，如：

ls：列出目录中的文件和子目录。

touch：创建文件。如touch 2.txt。

pwd：显示当前工作目录的完整路径。

mkdir：创建新目录。mkdir test_w/src -p ，如果要创建多级目录，需要加-p。

rm：删除文件或目录。

  • rm new_directory
  • 如果要删除的是个目录，需要加-r，如rm ccc -r。

cp：复制文件或目录。

  • cp 3.txt ~/bbb/ccc/，~代表根目录，将当前目录的3.txt文件复制到ccc文件目录下。
  • cp ~bbb/ccc/ ./ -r将ccc整个文件夹复制到当前目录下。

mv：移动文件或目录。

  • mv ~/bbb/ccc/4.txt ./将~/bbb/ccc内的4.txt移动当前目录下。
  • 如果后面一个文件夹不存在，则意味着重命名，mv 3.txt 5.txt将3.txt重命名为5.txt。
  • 如果目标文件夹非空，则无法移动，除非加-b，如mv ~/bbb/ccc/ ./ -b。

2. vim

输入i，可以编辑文件，按esc退出编辑模式，然后:q!退出，:wq保存并退出。

  • vim xxx.txt 如果 xxx.txt 文件存在则编辑，不存在则自动新建文件并编辑。
  • 刚进去编辑界面时，还处于非编辑模式，此时可按 i 键进入编辑模式。
  • 在 vim 界面中，只能在光标处进行编辑，光标的位置只能用方向键（上下左右）控制。
  • 当编辑完成需要保存时，按 Esc 键退出编辑模式，输入 :wq 并回车 ( 或者连按 2 次大写 Z)
  • 当需要放弃编辑时，按 Esc 键退出编辑模式，输入 :q! 并回车
  • 在非编辑模式下，按 u 可以撤销一次改动，按 ctrl+r 取消上一次撤销操作，按 dd 可以删除一行，按 yyp 可以复制一
    行。
  • 在非编辑模式下，按 / 再输入关键字并回车，可以搜索关键字，按 n 跳到出现关键字的下一个位置，按 N 跳到出现
    关键字的上一个位置。
  • 在非编辑模式下，按 gg 可使光标跳到第一行，按 G 可使光标跳到最后一行。
  • 在非编辑模式下，按 ~ 可以使光标处的字母进行大小写转换
  • 编辑系统文件时，需要加上 sudo ，比如 sudo vim ~/.bashrc

3. 其他

ifconfig：查看系统ip信息。

hostname：查看主机名，主机名通常作为网络通信中的唯一标志名。

sudo reboot：重启。

sudo chmod 777 xxx：给某文件最高权限。

  • sudo chmod 777 4.txt ，给4.txt最高权限。
  • 如果当前的是文件夹，那么不会给其子文件最高权限，除非最后加-R，如：sudo chmod 777 ccc/ -R。

ping ip或者ping 网址。

ssh xxx@ip。如ssh eaibot@192.168.31.200远程登录用户名为eaibot且ip为192.168.31.200的系统。

exit退出当前终端或远程登录。

clear清空屏幕内容。

history -c清空命名记录。

nl xxx查看xxx文件内容（显示行号），等于cat xxx -n（包含空行）。

ps -aux | grep xxx 显示名称带xxx的所有进程。然后kill 进程号杀死该进程。

top显示所有进程状态。

unzip xxx.zip 解压 xxx.zip 包到当前目录 ; tar -xzvf xxx.tar.gz 解压 xxx.tar.gz 包到当前目录 ; gunzip xxx.gz 解压 xxx.gz 到当前目录。
